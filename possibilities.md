# Self-Hosted Tools for Developers & Product Managers

## Currently Running
- **Excalidraw** - Whiteboard and sketching tool (port 3030)
- **Mermaid Live Editor** - Diagram-as-code editor (port 4040)

## Project Management
- **Plane** - Modern Jira alternative (Linear-like interface)
- **Focalboard** - Trello/Notion alternative by Mattermost
- **Taiga** - Agile project management
- **Wekan** - Kanban board
- **Planka** - Another Trello alternative

## Documentation & Knowledge Base
- **Outline** - Beautiful wiki/knowledge base (Notion-like)
- **BookStack** - Organized documentation platform
- **Wiki.js** - Modern wiki software
- **Docusaurus** - Documentation sites (static)
- **Foam** or **Logseq** - Obsidian Publish alternatives

## Note-taking & Writing
- **HedgeDoc (CodiMD)** - Collaborative Markdown editor
- **Trilium Notes** - Hierarchical note-taking
- **Joplin Server** - Sync server for Joplin notes

## Development Tools
- **Gitea/Forgejo** - Lightweight GitHub alternative
- **code-server** - VS Code in the browser
- **Jupyter** - Interactive notebooks
- **n8n** - Workflow automation (Zapier alternative)
- **Appsmith/Budibase/Tooljet** - Internal tool builders

## Monitoring & Analytics
- **Uptime Kuma** - Uptime monitoring
- **Plausible** - Privacy-friendly analytics
- **Grafana + Prometheus** - Metrics and dashboards

## Communication
- **Mattermost/Rocket.Chat** - Slack alternatives
- **Jitsi Meet** - Video conferencing
- **Element** - Matrix chat client

## Databases & Admin
- **Adminer/pgAdmin** - Database management UIs
- **Redis Commander** - Redis UI
- **Mongo Express** - MongoDB UI

## Productivity
- **Vikunja** - Todo lists and project management
- **Habitica** - Gamified task management
- **Rallly** - Meeting poll scheduler (Doodle alternative)

## Additional Diagramming Tools
- **tldraw** - Open-source infinite canvas
- **Drawio (diagrams.net)** - Feature-rich diagramming
- **PlantUML** - UML diagrams from text
- **D2** - Modern declarative diagram language
- **Graphviz/DOT** - Classic graph visualization
- **Kroki** - Unified API for multiple diagram-as-code tools
- **Markmap** - Mind maps from Markdown
- **ASCIIFlow** - Simple ASCII diagram drawing
